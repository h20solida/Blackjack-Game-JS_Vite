# BlackJack Vite

Pasos para ejecutar proyecto:

1. Clonar repositorio
2. Ejecutar `npm install` para reconstruir los módulos de node
3. Correr el devServer `npm run dev`
4. Abrir el proyecto en: `http://localhost:5173`

## Producción

1. Ejecutar `npm run build`
2. Tomar la carpeta `dist` y desplegarla

## 🛠 **Instalación Local**

git clone https://github.com/h20solida/js-vite-blackjack-main.git
cd js-vite-blackjack-main
npm install
npm run dev

# 🎰 **Blackjack Vite**

## Juego completo de Blackjack en Vanilla JS + Vite

## ✨ **Características**

- 🎴 **52 cartas reales** (PNG calidad casino)
- ♠️♥️♦️♣️ **4 palos completos**
- 🎯 **Lógica Blackjack completa** (21 puntos)
- 📱 **Responsive** móvil/tablet/PC
- ⚡ **Vite v8.0.2** (0 rebuilds)
- 🎨 **CSS profesional** mesa + cartas

## 🚀 **Como jugar**

1. "Pedir Carta" te ofrecerá el mismo número de cartas que de veces pulses el botón
2. Si te quieres plantar en una putuación "Detener" ; Recuerda que la máxima puntuación es de 21 puntos
3. "Jugador Gana" o "Computadora Gana" ; para comenzar nueva partida sería pulsar "Nuevo Juego"
